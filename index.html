How I did it
  • git init - for initialization in the project
  • git config --local user.name "OlehKuts" - or global (instead local) for all future projects
  • git config --local user.email - oleh.kuts@yahoo.com - email definition(without in quotes)
  • git status - may use for checking status (untracked - indexed(added(for primary) or modified(for old files) - committed))
  • git add -A - adding new files to pull (added)
  • git commit -a -m"my first commit in this project" - m"info about this commit"

Second commit has been created

  • git log - check info in terminal about all previous commits
  • git remote add origin https://github.com/OlehKuts/Git_basics.git - copy from the new repository on GitHub - this move allows to tie local and github repositories
  • git push -u origin main - now we can add changes to remote github repository too
